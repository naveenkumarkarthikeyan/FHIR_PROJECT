FHIR_Project

This repository demonstrates a complete **FHIR interoperability pipeline** using HL7 FHIR R4 standards. It includes:

- Dockerized deployment of a **HAPI FHIR R4 server**  
- Custom FHIR profiles created with **FHIR Shorthand (FSH) + SUSHI**  
- A **Python-based translator** that converts legacy CSV data into valid FHIR resources  
- Detailed documentation for mapping, architecture, and data flow
